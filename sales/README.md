# AI Pathshala Sales Toolkit

Complete sales resources for founder-led sales targeting Affordable Private Schools in Tier 2/3 Indian cities.

---

## Quick Start (Day 1)

1. **Read the 90-day plan:** `tracking/90-day-action-plan.md`
2. **Set up tracking:** Copy `tracking/sales-pipeline.md` to Google Sheets
3. **Prepare materials:** Review `materials/one-pager-brochure.md`
4. **Learn scripts:** Practice `scripts/whatsapp-outreach.md`
5. **Start outreach:** Research 10 schools and send first messages

## Target Market Summary

### Primary Target: Affordable Private Schools (APS)

| Attribute | Target |
|-----------|--------|
| Annual fees | Rs 5,000 - 50,000 |
| Student count | 200 - 1,000 |
| Location | Tier 2/3 cities |
| Board | CBSE, State Board |
| Language | Hindi medium, English medium |

**Phase 1 (Weeks 1-4):** Bihar

**Phase 3 (Weeks 9-12):** Expansion
- Jaipur, Indore, Bhopal

## Support & Questions

All sales materials are designed for a bootstrapped, founder-led sales approach. The focus is on:

1. **Speed over perfection** - Get to first customer fast
2. **WhatsApp-first** - Most school owners prefer WhatsApp
3. **Hindi-friendly** - All scripts work in Hindi/Hinglish
4. **ROI-focused** - Every pitch ties back to value
