# AI Pathshala Sales Pipeline Tracker

## Pipeline Stages

1. **Lead** - School identified, contact found
2. **Contacted** - First message sent
3. **Responded** - They replied (interested or question)
4. **Demo Scheduled** - Demo date set
5. **Demo Done** - Demo completed
6. **Trial** - Free trial started
7. **Negotiation** - Discussing pricing/terms
8. **Won** - Paid customer
9. **Lost** - Not converting (note reason)

## Key Metrics Definitions

- **Lead** = School name + city identified
- **Contact** = Phone/WhatsApp message sent
- **Response** = Any reply (even negative)
- **Demo** = Full 15-min demo conducted
- **Trial** = School actively using product
- **Won** = First payment received
- **MRR** = Monthly Recurring Revenue
- **Churn** = Customer stopped paying
