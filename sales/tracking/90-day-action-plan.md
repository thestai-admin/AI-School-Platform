## EMERGENCY TROUBLESHOOTING

### Not Getting Responses?
- [ ] Try voice notes instead of text
- [ ] Call directly (some prefer calls)
- [ ] Try different times (10 AM, 6 PM best)
- [ ] Shorten your message
- [ ] Lead with problem, not product

### Demos Not Converting?
- [ ] Ask for feedback: "What would make this useful?"
- [ ] Identify real objection
- [ ] Lower barrier: smaller plan, shorter trial
- [ ] Visit in person if possible

### Trials Not Using Product?
- [ ] Daily WhatsApp check-in
- [ ] Offer to create content FOR them
- [ ] Reduce scope: "Just try lesson generator"
- [ ] Address specific blocker

### Revenue Not Growing?
- [ ] Focus on customer success (reduce churn)
- [ ] Push referrals harder
- [ ] Expand to new cities
- [ ] Consider price increase if value is clear
